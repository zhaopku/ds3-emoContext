# emoContext
    SemEval 2019 task 3
## Requirements
    1. TensorFlow >= 1.10
    2. TensorFlow Hub
    3. tqdm
    4. sklearn
    5. nltk
    
## Commandline Options

    Please read models/train.py for details.

## Example Usage
    
    python3 main.py --universal --model hbmp+share --nLSTM 3 --hiddenSize 500 --dropOut 0.6 --preEmbedding --elmo --trainElmo --learningRate 0.001 --rnnLayers 1 --batchSize 100 --epochs 80
